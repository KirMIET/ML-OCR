# ML-OCR

Проект по распознаванию текста на изображениях с использованием глубокого обучения.

## Структура проекта

### `Ruslan/`
Ноутбуки и данные для экспериментов:
- `Ansamble.ipynb` - ансамблирование моделей
- `ClovaAI_CTC_*.ipynb` - Обучение моделей на базе ClovaAI
- `Create_syntetic.ipynb` - создание синтетических данных
- `data/` - вспомогательные данные
- `scr/` - скрипты для обработки данных

### `score 0.98936/`
Основной проект с лучшим результатом (точность 0.98936):
- `parseq/` - библиотека PARSeq модели
- `parseq_sint/`, `parseq_sint_crop/`, `parseq_swa/` - различные версии моделей
- `src/` - вспомогательные скрипты
- `inference_v*.py` - скрипты для инференса
